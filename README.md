# Olfah Cinematic V2

A cinematic Olfah property experience with a full-screen interaction model, complete project details, responsive gallery, enquiry flow and an interactive masterplan connected to eight 360° panoramas.

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

The V2 project is independent from the original Olfah landing page and reuses the approved Olfah asset library without altering the source renders.
